# 🤖 Binance グリッド取引ボット

Binance API を使用した自動グリッド取引ボットです。レンジ相場で効果を発揮し、小幅な値動きで利益を積み上げる戦略を採用しています。

## ⚠️ 免責事項

- **このボットは利益を保証するものではありません**
- 仮想通貨取引には高いリスクが伴います
- **必ず Binance Testnet で動作確認してから本番環境で使用してください**
- 実際の資金で取引する場合は、自己責任でお願いします

## ✨ 特徴

- ✅ **グリッド取引**: 自動的に買い注文と売り注文を配置
- ✅ **リスク管理**: 損切りライン、ポジション制限
- ✅ **リアルタイム監視**: CUI でステータス表示
- ✅ **Testnet 対応**: 安全にテスト可能
- ✅ **ログ記録**: 詳細な取引履歴を記録
- ✅ **PnL 追跡**: 実現利益・未実現利益を計算

## 🚀 セットアップ

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. Binance API キーの取得

1. [Binance](https://www.binance.com/) にログイン
2. API Management から API キーを作成
3. **スポット取引** を有効化
4. IP ホワイトリストを設定（推奨）

**Testnet 用 API キー**: https://testnet.binance.vision/

### 3. 環境変数の設定

```bash
# テンプレートをコピー
copy .env.example .env

# .env を編集
```

`.env` ファイル:
```env
# Binance API Credentials
BINANCE_API_KEY=your_api_key_here
BINANCE_API_SECRET=your_api_secret_here

# 取引設定
TRADING_SYMBOL=BTCUSDT
GRID_COUNT=10
LOWER_PRICE=
UPPER_PRICE=
INVESTMENT_AMOUNT=100

# リスク管理
STOP_LOSS_PERCENTAGE=5
MAX_POSITIONS=5

# Testnet 使用（true: テスト、false: 本番）
USE_TESTNET=true
```

## ⚙️ 設定

### 主要設定

| 設定項目 | 説明 | デフォルト | 備考 |
|---------|------|-----------|------|
| `TRADING_SYMBOL` | 取引ペア | BTCUSDT | Binance に存在するペア |
| `GRID_COUNT` | グリッド数 | 10 | 2 以上 |
| `LOWER_PRICE` | グリッド下限価格 | 自動（現在価格の -10%） | 空欄で自動計算 |
| `UPPER_PRICE` | グリッド上限価格 | 自動（現在価格の +10%） | 空欄で自動計算 |
| `INVESTMENT_AMOUNT` | 投資額（USDT） | 100 | 0 より大きい値 |
| `STOP_LOSS_PERCENTAGE` | 損切り割合（%） | 5 | 0-100 |
| `MAX_POSITIONS` | 最大ポジション数 | 5 | 1 以上 |
| `USE_TESTNET` | Testnet 使用 | true | **最初は true で** |

### グリッド数の設定

- **グリッド数が多い**: 小さな値動きで取引、1回あたりの利益は小さい
- **グリッド数が少ない**: 大きな値動きで取引、1回あたりの利益は大きいが機会減少

**推奨**: 20-50 グリッド（価格帯に応じて調整）

## 📊 グリッド取引戦略

### 仕組み

1. 価格帯を均等に分割（グリッド）
2. 各グリッドレベルに買い注文と売り注文を配置
3. 価格が下がれば買い、上がれば売る
4. グリッド間隔分の利益を積み上げる

### 例

```
価格範囲: $90,000 - $110,000
グリッド数: 10
間隔: $2,000

注文配置:
$110,000 ──────────── 売り注文
$108,000 ──────────── 売り注文
$106,000 ──────────── 売り注文
...
$92,000  ──────────── 買い注文
$90,000  ──────────── 買い注文
```

価格が $100,000 → $98,000 に下がると買い注文が約定  
価格が $98,000 → $100,000 に戻ると売り注文が約定  
利益: $2,000 × 数量

### 得意な相場

✅ **レンジ相場**（横ばい）: 効果を発揮  
⚠️ **トレンド相場**（急上昇・急下落）: 含み損が膨らむ可能性あり

## 🛡️ リスク管理

### 損切り

- 設定した割合で価格が下落した場合、全ポジションを解消
- デフォルト: 5%

### ポジション制限

- 同時に保有できるポジション数を制限
- デフォルト: 5

### 注意事項

- 急激な価格下落では、損切りがスリッページでずれる可能性あり
- グリッド範囲を大幅に下回る価格になると、すべての買い注文が約定する可能性

## ⚠️ 注意事項

1. **必ず Testnet でテストしてください**
2. 余剰資金で取引してください
3. 定期的にログを確認してください
4. ネットワーク接続が安定している環境で使用してください
5. API キーは絶対に他人と共有しないでください
6. 本番運用は小额から始めてください

## 🔧 トラブルシューティング

### エラー: `設定エラーがあります`

→ `.env` ファイルを確認してください。API キーが正しく設定されているか確認。

### エラー: `API リクエスト失敗`

→ API キーの権限を確認（スポット取引が有効か）  
→ IP ホワイトリストを確認

### 注文が約定しない

→ グリッド間隔が広すぎる可能性があります  
→ 価格がグリッド範囲外にある可能性があります

### ログの確認

```
logs/grid_bot_YYYYMMDD.log
```

詳細なログが記録されています。
